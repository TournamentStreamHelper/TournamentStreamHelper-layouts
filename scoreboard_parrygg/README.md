# Parry.gg Stream Layouts

- Includes game-specific overlays for **Super Smash Bros. Ultimate**, **Rivals 2**, and **Super Smash Bros. Melee**.
- Each layout features two different HTML files to accommodate different setups: one **with player cameras** and one **without player cameras**. You'll see them labled "nocams".
- Supports doubles
- If the tournament name is too long (ex. it has the pot bonus in the title), I recommend going to TSH "Tournament Info" tab and removing it to keep it looking nice.
